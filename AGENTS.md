# AGENTS.md

## 协作规则
- 在本项目中，任何代码改动前都必须先更新文档，再进入实现。
- “文档”至少包括 `README.md`；若协作规则发生变化，也必须同步更新本文件。
- 涉及页面交互、数据结构、目录结构、运行方式、脚本命令的修改，都要先把说明写清楚。

## 当前阶段门禁
- 当前只做项目初始化、文档建立与 mock 数据生成。
- 页面视觉设计、浏览分类、领养交互、每人最多领养三只等功能暂不开发。
- 只有在当前数据结构得到确认后，才能进入下一阶段页面开发。

## 数据约定
- 狗狗数据生成后存放于 `src/public/data/`。
- 本阶段同时维护 `dogs.csv` 与 `dogs.json` 两份产物。
- 数据字段变更时，必须先更新 `README.md` 中的数据说明，再修改脚本或类型定义。
